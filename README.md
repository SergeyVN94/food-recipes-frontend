# Описание

Это фронтенд часть сайта для поиска рецептов.

*Стек технологий*:

* React
* SCSS
* CSS modules
* Typescript

## Функционал сайта (в разработке)

* Возможность искать рецепты по списку ингридиентов, по тегам или категориям.
* Регистрация и личный кабинет
* Добавление рецептов в избранное в лк
* Добавление новых рецептов после регистрации
* Возможность разбить рецепты по шагам.
